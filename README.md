Project Description:

CLI Python Calculator Utility for the following operations:

1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Power
6. Root
7. Modulus
8. Integer Division
9. Percentage
10. Absolute
11. Absolute Difference

Installation Instructions: 

1. python -m venv venv 
2. source venv/bin/activate
3. pip install -r requirements.txt
4. python3 main.py

Testing Instructions

1. pytest

CI/CD Information

1. Execute Github Actions Workflow to run automated application setup and test coverage
